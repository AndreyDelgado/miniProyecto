# Aptify - Sistema de Búsqueda de Apartamentos

Este repositorio contiene el código del mini proyecto Aptify, desarrollado para el curso ISW-521 Programación en Ambiente Web I de la Universidad Técnica Nacional.

## Descripción del Proyecto

Aptify es una solución web interactiva en fase de desarrollo. Su objetivo principal es resolver la desorganización en la búsqueda de alquileres, permitiendo a los usuarios filtrar opciones de apartamentos por provincia, cantón, precio y requisitos específicos.

Actualmente, el proyecto incluye la maquetación inicial, la navegación entre páginas y la lógica básica de autenticación utilizando almacenamiento local.

## Tecnologías Utilizadas

El proyecto se construye utilizando tecnologías estándar:
* HTML5
* CSS3
* JavaScript

## Estructura de Archivos

* `index.html`: Página principal y presentación del sistema.
* `acceso.html`: Vista para el inicio de sesión y registro de usuarios.
* `buscar.html`: Catálogo de propiedades interactivo.
* `publicar.html`: Formulario para el registro de nuevos inmuebles.
* `css/style.css`: Hoja de estilos principal del sitio.
* `js/auth.js`: Lógica para el manejo de sesiones y validación de usuarios.
